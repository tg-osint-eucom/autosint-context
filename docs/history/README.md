This directory preserves tracked historical records whose current replacement
authority has been proven. Archived material is provenance, not current
architecture, workflow, implementation, runtime, proof, or release authority.

## Sections

- [`superseded/`](superseded/): superseded design and discovery records.
- [`incidents/`](incidents/): future sanitized incident records that are safe to
  track; none were moved in Production Foundation V2.1 Phase 13.
- [`migrations/`](migrations/): future historical migration records; none were
  moved in Production Foundation V2.1 Phase 13.

## Archive Gate

A tracked item may move here only when its lifecycle classification is current,
replacement authority exists, every tracked inbound reference is rebound, its
historical meaning is preserved, and focused documentation/link tests pass.
Archive placement never upgrades an old claim to current truth.

Deletion is a separate gate. Phase 13 deleted no tracked item because no item
was classified `DEAD_CONFIRMED`. Ignored runtime history, artifacts, receipts,
logs, screenshots, databases, raw Evidence, browser state, and private operator
history are outside this archive and were not moved or deleted.
